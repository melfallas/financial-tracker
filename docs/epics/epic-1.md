# Epic 1: Core Foundations & Technical Scaffold

**Goal:** Establish the Angular 21 environment, Tailwind v4 design system with PrimeNG integration (`tailwindcss-primeui`), Storybook configuration, Chart.js + ng2-charts setup, and the foundational "Wealth Gap" mathematical engine following strict TDD.

## Context
This epic sets up the structural foundation of the project, including the base configuration for Angular, styles, UI libraries, component documentation, testing and core pure-TypeScript logic for financial calculations.

## Stories Include:
- US1.1: Project Initialization (Architectural Scaffold)
- US1.2: Global Design System (Tokens, Variables & PrimeNG Theme)
- US1.3: Storybook Setup & Component Documentation
- US1.4: The Wealth Gap Engine (TDD Flow)
- US1.5: Repository Pattern & Local Persistence
