🛡️ La Realidad Técnica: No hay "Secrets" en el Cliente
Es fundamental entender este principio: En una aplicación de Frontend puro (estática), no existen los secretos. Aunque GitHub permite guardar Secrets en el repositorio, estos solo son accesibles durante el proceso de Build en GitHub Actions. Una vez que la aplicación se compila y se despliega en GitHub Pages, cualquier variable que el código necesite para funcionar (como una API Key) será visible para cualquier usuario que abra la consola del navegador (F12) y revise los archivos .js.

🛠️ Estrategia de Seguridad para Financial Tracker
Para manejar los tokens de las APIs de divisas e inversión de forma segura en este entorno, implementaremos estas 3 capas de protección:

1. Inyección en Tiempo de Build (Build-time Injection)
Usaremos los GitHub Secrets para guardar las llaves de producción. El Pipeline de GitHub Actions las tomará y las inyectará en el archivo de configuración de Angular (src/environments/environment.prod.ts) justo antes de compilar.

Ventaja: Las llaves no están en el código fuente del repositorio (público o privado).

Riesgo: Una vez desplegado, la llave es visible en el código minificado.

2. Restricción por Dominio (Domain Whitelisting) - CRÍTICO
Esta es nuestra principal defensa. Configuraremos los proveedores de API (ej. la API de tasas de cambio o la de datos del S&P 500) para que solo acepten peticiones que provengan de tu dominio: tu-usuario.github.io.

Si alguien roba tu API Key e intenta usarla desde su propia computadora o desde otro sitio web, el servidor de la API rechazará la petición automáticamente.

3. Uso de un Proxy "Serverless" (Opcional pero Recomendado)
Si alguna de las APIs de inversión que elijas es extremadamente sensible o costosa y no permite restricción por dominio, la solución es:

GitHub Actions + GitHub Pages no pueden ocultar la llave.

Solución: Usar una Vercel Function o Netlify Function gratuita como intermediario (Proxy). Tu app de Angular llama a la función (sin llaves visibles), y la función llama a la API usando la llave oculta en el backend del proveedor de serverless.

1. ¿Soporta Supabase la Restricción por Dominio?
Sí, pero de una forma específica. Supabase no bloquea la API Key (la anon_key) basándose en el dominio de origen de la petición HTTP de la misma manera que lo hace una API de Google Maps o ExchangeRate. En Supabase, la anon_key es, por diseño, pública y se puede ver en el frontend.

2. La verdadera defensa: Row Level Security (RLS)
En lugar de depender solo del dominio, la seguridad en Supabase se basa en RLS. Esto significa que:

Aunque alguien robe tu anon_key e intente consultar tu base de datos desde su computadora, no podrá ver ni modificar ningún dato a menos que tú hayas creado una política que lo permita explícitamente.

3. Configuración de CORS en Supabase
Supabase permite configurar los dominios permitidos (CORS) en su panel de control.

Podremos añadir https://tu-usuario.github.io a la lista de dominios permitidos.

Resultado: Si alguien intenta hacer una petición fetch a tu instancia de Supabase desde otro dominio (ej. hacker-site.com), el navegador bloqueará la petición por política de CORS.

4. Implementación Local Temporal (src/app/infrastructure/local-storage.service.ts)
Esta clase cumple con la interfaz pero guarda los datos únicamente en el navegador.

TypeScript
import { Injectable } from '@angular/core';
import { ILeadRepository } from '../core/interfaces/i-lead-repository';
import { Lead } from '../shared/types';
import { Observable, of } from 'rxjs';

@Injectable({ providedIn: 'root' })
export class LocalStorageService implements ILeadRepository {
  private readonly STORAGE_KEY = 'ft_leads_db';

  saveLead(lead: Lead): Observable<boolean> {
    const existing = this.getLocalData();
    const newLead = { ...lead, id: Crypto.randomUUID(), createdAt: new Date() };
    
    localStorage.setItem(this.STORAGE_KEY, JSON.stringify([...existing, newLead]));
    console.log('📦 Lead guardado en LocalStorage:', newLead);
    return of(true);
  }

  getLeads(): Observable<Lead[]> {
    return of(this.getLocalData());
  }

  private getLocalData(): Lead[] {
    const data = localStorage.getItem(this.STORAGE_KEY);
    return data ? JSON.parse(data) : [];
  }
}

export interface ProjectionResult {
  year: number;
  totalBalance: number;
  contributions: number;
  interestEarned: number;
}

/**
 * Calcula la proyección de interés compuesto año a año.
 */
export const calculateCompoundInterest = (
  initialAmount: number,
  monthlyContribution: number,
  annualRate: number,
  years: number
): ProjectionResult[] => {
  const results: ProjectionResult[] = [];
  let totalBalance = initialAmount;
  let totalContributions = initialAmount;
  const monthlyRate = annualRate / 100 / 12;

  for (let year = 1; year <= years; year++) {
    for (let month = 1; month <= 12; month++) {
      const monthlyInterest = totalBalance * monthlyRate;
      totalBalance += monthlyInterest + monthlyContribution;
      totalContributions += monthlyContribution;
    }
    
    results.push({
      year,
      totalBalance: Math.round(totalBalance),
      contributions: Math.round(totalContributions),
      interestEarned: Math.round(totalBalance - totalContributions)
    });
  }
  
  return results;
};

export interface ProjectionResult {
  year: number;
  totalBalance: number;
  contributions: number;
  interestEarned: number;
}

/**
 * Calculates compound interest projection on a yearly basis.
 * Formula: A = P(1 + r/n)^(nt) + PMT * [((1 + r/n)^(nt) - 1) / (r/n)]
 */
export const calculateCompoundInterest = (
  initialAmount: number,
  monthlyContribution: number,
  annualRate: number,
  years: number
): ProjectionResult[] => {
  const results: ProjectionResult[] = [];
  let totalBalance = initialAmount;
  let totalContributions = initialAmount;
  const monthlyRate = annualRate / 100 / 12;

  for (let year = 1; year <= years; year++) {
    for (let month = 1; month <= 12; month++) {
      const monthlyInterest = totalBalance * monthlyRate;
      totalBalance += monthlyInterest + monthlyContribution;
      totalContributions += monthlyContribution;
    }
    
    results.push({
      year,
      totalBalance: Math.round(totalBalance),
      contributions: Math.round(totalContributions),
      interestEarned: Math.round(totalBalance - totalContributions)
    });
  }
  
  return results;
};

@Component({
  selector: 'app-compound-interest',
  standalone: true,
  imports: [CommonModule, FormsModule],
  templateUrl: './compound-interest.html'
})
export class CompoundInterest {
  // Input Signals for calculation
  initialAmount = signal(5000);
  monthlyContribution = signal(500);
  annualRate = signal(10);
  years = signal(20);

  // Reactive projection result using computed signals
  projection = computed<ProjectionResult[]>(() => {
    return calculateCompoundInterest(
      this.initialAmount(),
      this.monthlyContribution(),
      this.annualRate(),
      this.years()
    );
  });

  // Final total balance signal
  finalBalance = computed(() => {
    const results = this.projection();
    return results.length > 0 ? results[results.length - 1].totalBalance : 0;
  });
}

1. Actualización de la Lógica de la Calculadora (src/app/features/compound-interest/compound-interest.ts)
Integramos el CurrencyService y transformamos los resultados usando computed signals.

TypeScript
import { Component, signal, computed, inject } from '@angular/core';
import { CommonModule } from '@angular/common';
import { FormsModule } from '@angular/forms';
import { CurrencyService } from '../../core/services/currency.service';
import { calculateCompoundInterest, ProjectionResult } from '../../shared/utils/finance-math';

@Component({
  selector: 'app-compound-interest',
  standalone: true,
  imports: [CommonModule, FormsModule],
  templateUrl: './compound-interest.html'
})
export class CompoundInterest {
  private currencyService = inject(CurrencyService);

  // Input Signals (Always kept in USD internally for stability)
  initialAmountUsd = signal(5000);
  monthlyContributionUsd = signal(500);
  annualRate = signal(10);
  years = signal(20);

  // Reactive access to the selected currency metadata
  public selectedCurrency = computed(() => {
    const code = this.currencyService.selectedCurrency();
    return this.currencyService.currencies().find(c => c.code === code);
  });

  // Projection calculated in USD, then mapped to the local currency
  projection = computed<ProjectionResult[]>(() => {
    const usdResults = calculateCompoundInterest(
      this.initialAmountUsd(),
      this.monthlyContributionUsd(),
      this.annualRate(),
      this.years()
    );

    const rate = this.currencyService.currentRate();

    // Convert results to the selected currency (EUR, CRC, ARS, etc.)
    return usdResults.map(res => ({
      ...res,
      totalBalance: res.totalBalance * rate,
      contributions: res.contributions * rate,
      interestEarned: res.interestEarned * rate
    }));
  });

  finalBalance = computed(() => {
    const results = this.projection();
    return results.length > 0 ? results[results.length - 1].totalBalance : 0;
  });
}

1. Lógica Matemática Ajustada (src/app/shared/utils/finance-math.ts)
Adding the inflation adjustment calculation to the core engine.

TypeScript
export interface ProjectionResult {
  year: number;
  nominalBalance: number;  // Growth without inflation
  realValueBalance: number; // Growth adjusted for purchasing power
  interestEarned: number;
}

/**
 * Calculates compound interest vs inflation-adjusted value.
 * real_value = nominal_value / (1 + inflation_rate)^years
 */
export const calculateAdvancedProjection = (
  initialAmount: number,
  monthlyContribution: number,
  annualRate: number,
  inflationRate: number,
  years: number
): ProjectionResult[] => {
  const results: ProjectionResult[] = [];
  let nominalBalance = initialAmount;
  const monthlyRate = annualRate / 100 / 12;

  for (let year = 1; year <= years; year++) {
    for (let month = 1; month <= 12; month++) {
      const monthlyInterest = nominalBalance * monthlyRate;
      nominalBalance += monthlyInterest + monthlyContribution;
    }
    
    // Purchasing power adjustment
    const inflationAdjustmentFactor = Math.pow(1 + (inflationRate / 100), year);
    const realValueBalance = nominalBalance / inflationAdjustmentFactor;

    results.push({
      year,
      nominalBalance: Math.round(nominalBalance),
      realValueBalance: Math.round(realValueBalance),
      interestEarned: Math.round(nominalBalance - (initialAmount + (monthlyContribution * 12 * year)))
    });
  }
  
  return results;
};

📈 3. Lógica Financiera: El "Wealth Gap" Engine
src/app/shared/utils/finance-math.ts
Esta utilidad pura maneja el cálculo de interés compuesto ajustado por inflación.

TypeScript
/**
 * Advanced financial projection including inflation impact.
 * Formula: Real Value = Nominal / (1 + inflation)^years
 */
export const calculateAdvancedProjection = (
  initial: number,
  monthly: number,
  rate: number,
  inflation: number,
  years: number
) => {
  let nominal = initial;
  const monthlyRate = rate / 100 / 12;
  const data = [];

  for (let y = 1; y <= years; y++) {
    for (let m = 1; m <= 12; m++) {
      nominal += (nominal * monthlyRate) + monthly;
    }
    const realValue = nominal / Math.pow(1 + inflation / 100, y);
    data.push({ year: y, nominal: Math.round(nominal), real: Math.round(realValue) });
  }
  return data;
};

/**
 * Advanced financial projection.
 * Adjusted for local inflation based on currency selection.
 */
export const calculateProjection = (initial: number, rate: number, inflation: number, years: number) => {
  const nominal = initial * Math.pow(1 + rate/100, years);
  const realValue = nominal / Math.pow(1 + inflation/100, years);
  return { nominal, realValue };
};

2. The Math Engine (Wealth Gap)
File: src/app/shared/utils/finance-math.ts
Purpose: Pure functions for calculation, decoupled from the UI.

TypeScript
export interface ProjectionEntry {
  year: number;
  nominalBalance: number;
  realValue: number; // Adjusted for inflation
}

/**
 * Calculates growth adjusted for purchasing power loss.
 * Logic: RealValue = Nominal / (1 + inflation)^year
 */
export const calculateWealthGap = (
  initial: number,
  monthly: number,
  annualRate: number,
  inflation: number,
  years: number
): ProjectionEntry[] => {
  const result: ProjectionEntry[] = [];
  let currentNominal = initial;
  const monthlyRate = annualRate / 100 / 12;

  for (let y = 1; y <= years; y++) {
    // Monthly compounding
    for (let m = 1; m <= 12; m++) {
      currentNominal += (currentNominal * monthlyRate) + monthly;
    }
    
    // Inflation adjustment
    const inflationFactor = Math.pow(1 + (inflation / 100), y);
    const currentReal = currentNominal / inflationFactor;

    result.push({
      year: y,
      nominalBalance: Math.round(currentNominal),
      realValue: Math.round(currentReal)
    });
  }
  return result;
};

@Injectable({ providedIn: 'root' })
export class CurrencyService {
  // Global Signals
  public selectedCurrency = signal<string>('USD');
  public fxRate = signal<number>(1.0); // Exchange rate vs USD
  public localInflation = signal<number>(3.4); // Current country inflation

  // Formatted display signal
  public currencyLabel = computed(() => `Currency: ${this.selectedCurrency()}`);

  updateCurrency(code: string, rate: number, inflation: number) {
    this.selectedCurrency.set(code);
    this.fxRate.set(rate);
    this.localInflation.set(inflation);
  }
