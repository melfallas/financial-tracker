# 📑 Context Snapshot: Financial Tracker (MVP)

## 1. Core Mission & Strategy

- **Project Name:** Financial Tracker
- **Product Goal:** High-conversion lead generation tool for the fintech sector.
- **Unique Selling Point (USP):** Visualizing the "Wealth Gap"—the divergence between nominal compound interest and real purchasing power (inflation-adjusted).
- **Conversion Engine:** Value-exchange model (User provides Name/Email in exchange for a personalized PDF Investment Report).

## 2. Technical Stack (The "Winston" Blueprint)

- **Frontend:** Angular 21 (Zoneless, Signal-based reactivity).
- **Styling:** Tailwind CSS v4 (Using CSS variable design tokens).
- **Architecture Pattern:** **BMad** (Base, Modals, Assets, Data).
- `Base`: Layout, Navigation, Charts.
- `Modals`: Lead Capture, Privacy Policies.
- `Assets`: i18n (EN/ES), Icons, Design Tokens.
- `Data`: Math Engine, Repository Services, API integration.

- **Persistence:** Repository Pattern (Phase 1: `LocalStorage` | Phase 2: `Supabase`).
