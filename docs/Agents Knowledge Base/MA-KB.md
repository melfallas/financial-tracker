# 🧠 Unified Knowledge Base: Financial Tracker

#### 1. Project DNA (The Strategic Core)

- **Mission:** To provide a premium financial utility that exposes the impact of inflation on savings to generate high-quality leads.
- **The "Aha!" Moment:** The visual revelation of the **Wealth Gap**.
- **Conversion Strategy:** Value-Exchange (Personalized PDF Report for User Data).

#### 2. Technical Pillars (The Winston Layer)

- **Architecture:** **BMad** (Base, Modals, Assets, Data).
- **Framework:** **Angular 21** (Signal-based, zone-less reactivity).
- **Styling:** **Tailwind CSS v4** (CSS variables as tokens).
- **Persistence:** **Repository Pattern** (Decoupling UI from Storage).

#### 3. Visual Identity (The Sally Layer)

- **Branding:** Strictly **Financial Tracker**.
- **80/15/5 Rule:**
- **Deep Blue (#1A3C6E):** Stability and Trust (80%).
- **Teal Bright (#009688):** Interaction (15%).
- **Emerald Green (#00C853):** Conversion Focus (5%).

- **Typography:** Inter (Sans-serif) for high readability in data-heavy views.

#### 4. Mathematical Logic (The Data Layer)

- **Core Calculation:** Monthly compounding interest adjusted by annual inflation.
- **Variables:**
- : Initial Principal.
- : Monthly Contribution.
- : Annual Interest Rate.
- : Annual Inflation Rate.
- : Number of Years.

#### 5. User Journey (The John Layer)

- **Input:** User sets parameters via sliders.
- **Impact:** Chart displays the divergence of Nominal vs. Real value.
- **Action:** User clicks "Download Report."
- **Capture:** Modal collects lead info.
- **Deliver:** PDF is generated and downloaded.
